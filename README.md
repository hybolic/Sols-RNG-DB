# Sols RNG DB

Database of Sols RNG Things

##

### WORK IN PROGRESS

This repo database is a work in progress of all the things we have collected about sols rng to be pressented as an easy to grab json file for applications, macros or discord bots.

### CONTRIBUTION

if you want to make an addition please have sources for your contribution!
eg. if a new potion is released this would be the example of the data we'd need

- a screenshot of the potion item description in the inventory
- a screenshot of the effect or effects description on the potion bar at the bottom of the screen given by item
- name of potion item
- name of the effect or effects
- the potion time given when item used

idealy we'd like this done with a git pull request with the given data attached as a comment but if this is not possible open an issue with the same information attached
